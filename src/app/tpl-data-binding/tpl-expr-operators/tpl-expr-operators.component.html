Template expression operators, Operadors de expressiones de plantillas

Pipe Operator (|)

title | uppercase

{{ title | uppercase }}

Puedes encadenar los pipes

title | uppercase | lowercase

{{title | uppercase | lowercase }}

Les puedes pasar parametros a los pipes

person?.fechaNacimiento | date: 'longDate'

{{ person?.fechaNacimiento | date: 'longDate' }}

El pipe json es muy util

person | json

{{person | json}}

Safe navigation operator (.?) and null properties path, Operator de proteccion de navegacion y propiedades nullas

Cuando alguna proiedad o subpropiedad es null, javascript dispara un error y angular tambien. Este error hace que la aplicacion completa se detenga y no muestre nada .

Podemos utilizar la directiva *ngIf y operadores de javascript como &&, pero cuando es una propiedad muy profunda el codigo se veria demasiado engorroso.

Por lo anterior existe el operador de proteccion en la navegacion

En este ejemplo si person es null, o hijos es null, la applicacion sigue funcionando y deja en blanco el valor de null

person?.hijos?.nombre

Primer hijo = {{person?.hijos?.nombre}}

Ahora accedindo al segundo hijo por error

person?.hijos[1]?.nombre

Primer hijo = {{person?.hijos[1]?.nombre}}

Ahora accedindo al primer hijo correctaente

person?.hijos[0]?.nombre

Primer hijo = {{person?.hijos[0]?.nombre}}

The Non-null assertion operator (!)

Este operador es util cuando enciendes la bandera --strickNullChecks, la cual obliga a toda tu aplicacion a no tener algunar referencia null o undefined en variables y propiedades

Utilizando el operator de afirmacion no nulla, no te protege de valores null o undefined, pero suspende el chequeo stricto de nulls, abajo esta un ejemplo de su sintaxis:

person.herramientas!.algo

herramienta.algo = {{person.herramientas!.algo}}
