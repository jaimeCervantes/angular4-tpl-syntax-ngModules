Event Binding, Enlazado de eventos (event)

Target Event

Se puede utilizar el prefijo on-, conocido como forma canonica

Al igual que pasa con las property bindings, angular revisa primero si existe alguna directiva que tiene definido algun target event, y despues los demas event targets de elementos, por ejemplo:

<!-- `myClick` is an event on the custom `ClickDirective` -->
<div (myClick)="clickMessage=$event" clickable>click with myClick</div>

$event y sentencias para el manejo de eventos

En este ejemplo se enlaza la propiedad value con person.name(Property binding), y ademas se bindeo el evento 'input' de los elementos inputs (event binding), estableciendo person.name = $event.target.value

<input type="text" [value]="person.name" (input)="person.name = $event.target.value">

Si el evento pertenece a una directiva( Recuerda que una directiva es un component), el objeto $event sera lo que definia ese directiva,y puede ser diferente en cada directiva y/o componente
