Input and output properties (@Input() and @Output())

Necesitamos saber la diferencia entre target binding y source binding

Target binding es la parte que esta a la izquierda del signo de =

Source binding es la parte a la derecha de =

Target binding se identifica por estar dentro de los signos [], () o [()]

Source binding se indentifica por dentro de "" o interpolacion

Cada miembro de la fuente de la directivve (tambien source binding) esta automaticamente disponible para binding. Osea no tienes que hacer nada especial para acceder a ellas dentro de un componente usando template expression(expresiones de plantilla) o template statements(sentencias de plantilla)

Se tiene acceso limitado a el objetivo del bindeo( targe binding), solo puedes hacer bindeos a las propiedades target que estan explicitamente definidas como inputs y ouputs usando los decoradores @Input("alias") y @Output("alias")

Source binding (sourrce members):

<img [src]="iconUrl"/> <button (click)="onSave()">Save</button>

Target binding:

<hero-detail [hero]="currentHero" (deleteRequest)="deleteHero($event)"></hero-detail>
