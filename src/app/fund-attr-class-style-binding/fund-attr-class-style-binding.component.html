Attributes, class and style bindings

Attributes

Este es la unica excepcion acerca de que los bindeos se hacen a los propiedades y no a los atributos

<td [attr.colspan]="1 + 1">Uno-Dos</td>

Uno-Dos
5 6

Para ARIA attributos

[attr.aria-label]="actionName"

Class binding

usando atributo normal class y [class]="classBinding"

[class.special]="isSpecial"

class="clase-normal" [class.special]="isSpecial" si isSpecial retorna true, entonces sera class="clase-normal special"

Style binding

Podemos utilizar camel-case o dash-case

Algunas bindeos de estilos permite establecer unidades

Es importante saber que style binding no es recomendado para difinar mas de un estilo, para estilo existe la directiva ngStyle o mejor aun, la directiva ngClass.
