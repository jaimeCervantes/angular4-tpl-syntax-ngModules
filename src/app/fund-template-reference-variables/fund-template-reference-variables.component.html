Template reference variables, variables de templates (#var)

Estas variables pueden guardar la referencia a un elemento del DOM, una directiva, un componente de angular o un component web nativo. Todo esto claro dentro del template donde es definido

Ejemplos:

<input type="text" #nombre> {{ nombre.value }}

Valor de la variable #nombre: {{nombre.value}}

Valor actual: {{currentValue}}

Normalmente una variable de este tipo guarda su valor igual al elemento que es aplicado, pero se le puede asignar otro valor, por ejemplo:

#personForm="ngForm"

{{submitMessage}}
